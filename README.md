# life-zoetrope-hypothesis

## equations

  $`
  \mathrm{III}_T(t)=\sum_{n\in\mathbb{Z}}\delta(t-nT)
  `$
  
* Apparent angular velocity（Forward / Pause / Reverse）：

  $`
  \omega_{\text{app}}=\omega_0-k\,\Omega,\quad \Omega=\tfrac{2\pi}{T}
  `$
  
  | Surplus | Direction |
  |-----|-----|
  |$>0$|Forward|
  |$=0$|Pause|
  |$<0$|Reverse|
  
* Flicker：

  $`
  T(t)=T_0(1+\eta(t)),\ \ |\eta|\ll1
  `$
